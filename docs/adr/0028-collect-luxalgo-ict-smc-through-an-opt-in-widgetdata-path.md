# Collect LuxAlgo ICT/SMC through an opt-in widgetdata path

LuxAlgo ICT/SMC is kept out of the default curated indicator allowlist and collected only through an explicit opt-in path. That path records the TradingView collection backend and uses `widgetdata` because public SMC/LuxAlgo studies can fail on the default TradingView chart backend while still returning usable graphics through widgetdata.
