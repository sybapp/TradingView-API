# Use LuxAlgo ICT/SMC as a candidate signal source

LuxAlgo ICT/SMC output is treated as a TradingView Data Source that can produce Structural Features and derived Candidate Signals, not as a strategy or final validation result. The first strategy design uses directional Structure Events such as BOS, CHoCH, and MSS as timing events, while order blocks and fair value gaps remain Liquidity Zones used for later touch or reclaim confirmation; all LuxAlgo-derived features start as repainting-risk until proven otherwise.
