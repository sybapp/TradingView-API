## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues using the `gh` CLI. External PRs are not a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage labels use the default canonical vocabulary. See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain docs layout. See `docs/agents/domain.md`.
